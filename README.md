Because the raw processed EEG dataset (**MindBigData-EP-v1.0.zip**) is approximately 2.6 GB, it exceeds GitHub's file storage limits and is not hosted directly in this repository. 

To run the source code or experiment notebooks locally, you must manually fetch and structure the data:

1. **Download the Dataset:** Visit the MindBigData open database at [https://mindbigdata.com/opendb/index.html](https://mindbigdata.com/opendb/index.html) and download the **MindBigData-EP-v1.0.zip** file.
2. **Create the Local Directory:** In the root folder of this cloned repository, create a new folder named exactly `dataset`.
3. **Place the Data:** Downloaded zip file and save the extracted data inside that new `dataset/` directory.
